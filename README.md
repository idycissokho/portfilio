- **Langage** : PHP 8

## 📁 Structure

```
resources/views/
├── layouts/
│   └── app.blade.php      # Layout principal (navbar + footer)
└── home.blade.php         # Page d'accueil (toutes les sections)

public/
└── moi.jpg                # Photo de profil
```

## ⚙️ Installation

```bash
git clone https://github.com/ton-username/portfilio.git
cd portfilio
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
```

Accède au portfolio sur `http://localhost:8000`
